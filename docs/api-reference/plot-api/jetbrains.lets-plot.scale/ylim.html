fun ylim(limits: Any): Feature

Specifies data range for y axis. Set limits if you want values to be consistent across multiple plots. Setting limits will remove data outside of the limits.

Parameters

limits - The data range for y axis. Continuous scale: a pair of numbers providing limits of the scale. Use null to refer to default min/max. Discrete scale: list of data values to display on axis, and their order.
